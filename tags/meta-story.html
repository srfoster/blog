Story/Meta-Story

2019-10-10 :: 2-minute read, story, meta-story

Consider learning that an ancient tome you found in a used bookstore was written as its author slowly went insane. Knowing this would change the experience of reading that book! I, for one, would like to know such meta-stories up front. Some might prefer to learn it afterward. But I think we can all agree that knowing the meta-stories at some point generally enhances the reading experience.
